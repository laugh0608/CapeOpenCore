Package Editor

1.0

(C) CO-LaN 2011: http://www.colan.org/ Implemented by AmsterCHEM 2011: http://www.amsterchem.com/

This project (PackageEditor) implements a utility to edit the collection of existing property packages.

Editing the collection of property package can be done by invoking Edit on the property package manager or thermo system as well, but not all CAPE-OPEN PMEs support such an action. Therefore, an application that does this is recommended.

All it does is make a single call into the IdealThermoModule DLL.
