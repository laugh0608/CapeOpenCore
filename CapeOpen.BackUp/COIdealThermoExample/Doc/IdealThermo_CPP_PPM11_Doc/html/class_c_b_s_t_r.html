Detailed Description

wrapper class for BSTR values

This class is for BSTR data manipilation. The main data type is a BSTR.

The BSTR is cleaned up at the destruction of this class.

Please take great care of knowing who owns the content of a BSTR and therefore who is responsible for freeing the content; this shows from [in], [in, out] and [out, retval] annotations in the IDL

Implementations in this class take into account that an empty string should be represented by NULL
