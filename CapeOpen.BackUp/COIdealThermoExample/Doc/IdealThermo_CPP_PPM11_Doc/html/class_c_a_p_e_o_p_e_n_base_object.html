Detailed Description

Base class for any derived CAPE-OPEN object.

Base class for any derived CAPE-OPEN object. Implements basic error handling and identification which every CAPE-OPEN object should implement. All CAPE-OPEN objects derive from this class

const OLECHAR *  name = NULL,
const OLECHAR *  description = NULL 
) [inline]

Constructor.

The base class needs to know whether the name and the description of the object can be changed by external users. Optionally, a name and description initializer can be passed with the constructor. The name of an object should not be empty, so if no name is given at time of construction, one should provide a name immediately after construction.

Parameters:
canRename should be true for classes of which the name and description may be changed by external applications.
name the initial name, optional
description the initial description, optional
