Detailed Description

Solver1Dim class.

Class for solving non-linear 1-dimensional problems

This is a simple solver solver that requires bracketing the solution. The solution will then be found by reducing the bracketed region based upon 1st order information (without the requirement to evaluate the derivative).

The solver presumes the function to be solved is monotonically increasing or decreasing
