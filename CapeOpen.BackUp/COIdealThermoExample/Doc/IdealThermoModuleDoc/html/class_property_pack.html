Detailed Description

PropertyPack class.

This is a wrapper class that access the PropertyPackage in a manner that is ok to expose from the DLL. External C++ client can use this class.

Calculate phase equilibrium.

Calculate vapor liquid phase equilibrium. The vaues are returned in arrays that are allocated and stored by this DLL. The return values are only valid until the next call to GetSinglePhaseProperties, GetTwoPhaseProperties or Flash, so store the return values, but not the pointers to them.

Parameters:
nComp Number of compounds in the mixture
compIndices Indices of the compounds in the mixture. One index for each compounds. Must be between 0 and number of compounds-1, inclusive
X Overall mole fractions[mol/mol], one value for each compound, assumed normalized
type Type of specifications passed (e.g. TP for a temperature and pressure specification)
spec1 Value of first specification (e.g. T/[K] for TP)
spec2 Value of second specification (e.g. P/[Pa] for TP)
phaseCount Receives the number of phases at equilibrium
phases Receives the types of the existing phases (Vapor or Liquid)
phaseFractions Receives the phase fractions of the existing phases [mol/mol]
phaseCompositions Receives the compositions of the existing phases [mol/mol]; one array for each phase, each array contains one mole fraction for each compound
T Receives the temperature at equilibrium
P Receives the pressure at equilibrium
Returns:
True if ok

Get single-phase mixture properties at specified temperature, pressure and composition.

Calculate and get single phase mixture properties. The properties are returned in arrays that are allocated and stored by this DLL. The return values are only valid until the next call to GetSinglePhaseProperties or GetTwoPhaseProperties, so store the return values, but not the pointers to them. Multiple properties can be requested in a single call. For each property, the values and number of values are returned

Parameters:
nComp Number of compounds in the mixture
compIndices Indices of the compounds in the mixture. One index for each compounds. Must be between 0 and number of compounds-1, inclusive
phaseID ID of the phase for which to calculate the properties
T Temperature [K]
P Pressure [Pa]
X Mole fractions [mol/mol], one value for each compound, assumed normalized
nProp Number of properties requested
propIDs IDs of the properties requested
valueCount Receives the number of values for each of the properties, one value for each property
values Receives the values, one double array for each property. Size of the array corresponds to valueCount for each property
Returns:
True if ok
