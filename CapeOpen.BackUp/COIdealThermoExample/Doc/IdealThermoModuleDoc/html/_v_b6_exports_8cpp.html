#define VBBOOL (   expr  )     ((expr)?VARIANT_TRUE:VARIANT_FALSE)

VB6 does not operate well on booleans that have a value of 1 for true, instead -1 is used. This causes funny things like res=True and Not res=True as well if not done properly as Not is a bitwise operator
